
ASCII编码大致可以分作三部分组成：

第一部分是：ASCII非打印控制字符（参详ASCII码表中0-31）;

第二部分是：ASCII打印字符，也就是CTF中常用到的转换;

第三部分是：扩展ASCII打印字符(第一第三部分详见 [ASCII码表](http://www.asciima.com/) 解释)。
